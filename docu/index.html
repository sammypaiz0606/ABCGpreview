Basic Information

SLAMR Features

Introduction - SLAMR is the Ultimate Resume and Personal Portfolio template that will serve all your needs. It is modern, clean, flexible, retina ready, responsive, uses parallax effects and much more. It is HTML5 & CSS3 Valid and tested on various devices. Use SLAMR to promote yourself, your work, your experience and show people what you have achieved in your life. Regardless what you need. SLAMR should have it.

Bootstrap 3 HTML5 CSS3 - SLAMR is built on top of the popular Bootstrap framework. Use all its features and build a bullet proof site in no time. Of course SLAMR incorporates HTML5 & CSS3 features.

Header Variations - Headers are the first thing a new user sees. So you need to catch his attention instantly. SLAMR provides many ways to help you do so. Use a video background in your header or a background slider. Optionally you can also use a basic background image / background color / background pattern. It is up to you. Subtile and state of the art parallax effects underline the professional look. On top of the background variations we have put 5 different header styles on top of this pack.

Installing

The installation is straight forward. Unzip the downloaded pack. You will find a template folder with all the files for this template.
Edit the files to your needs in your favourite editor.

Best Practice Advice

As this template comes with a lot of different styles and demos it's recommended to start with the most suiting demo.

Note: The template can also be displayed offline (however the contact form will not work as it needs PHP support).
After you have done your updates upload all the files to your server.

Checklist

  • Replace all the placeholder images
  • Replace all the dummy texts
  • Check all the links
    1. if there's not just a # sign
    2. if they actually go somewhere
    3. Remember target:_blank if you want to open up a new tab/window
  • Edit the contact form php file

Styles

Skins

SLAMR comes with two different skins. A light style (default) colorscheme-0.css and a flat style colorscheme-1.css. The flat style is really simple to activate. Just change the colorscheme-0.css to colorscheme-1.css. That's it. The file is located in the top part of your index

Header Variations

Various header variations are possible with SLAMR Have a look at the demo files.

Video Background

The video background takes the ID of a Youtube Video. Therefore go to the init section at the end of your file and search for SLAMR.videoBgYT. The last parameter is the ID of your video.

Background Slider

The images for the background slider can be set in the slides-container class of your file. Just include an image with an img tag. SLAMR will handle the rest. If you want to make modifications (e.g. on the sliding speed) go to SLAMR.headSlider in the init section of the file.

Static Image

Static image is very similar to the background slider. Just include one image (you can also include more. The first image will be taken though) in the slides-container class of your file. In the init section set SLAMR.headSlider to play:false in order to prevent sliding.

Background Pattern

The background pattern is set in the main.css file. Just set the body background:url to the pattern you would like. Also make sure that the slides-container class of your file is empty and does not include an image.

Background Color

Just give the #headid a background within the main.css this will set a background color.

Navbar Variatons

Choose between a navbar that is transparent at first and then becomes a background color or a navbar that always has a background color. Use the class static-background at the very beginning of the nav class to choose between the two styles.

The div gets the class animated and two data attributes. The first one defines the delay after which the animation will be triggered. Data-animation defines the animation.

Contact Form

Preconfigurations

For the contact form to work properly you have to make some changes.
First make sure that PHP is installed on your server.

Put this code in a new file and call it test.php. It is important that the file ending is *.php. Access test.php. If you can see the code you just put in this file than you have NOT installed PHP on your server. Otherwise you have PHP on your server.

Change email recipient

So assuming PHP is installed properly, what you have to do now is changing the email recipients within the contact.php (located in the inc folder). So all requests are received by you.
Search for the following line and replace YOUR.EMAIL@MAIL.COM with your email. Please make sure that your email is still within the quotes. 

							$admin_email = 'YOUR.EMAIL@MAIL.COM';

Test the changes

Test the functionality of the form by filling out the contact on your own.
